Football Player Profiles-UI
A small modern Flutter app that displays football players in a full-screen carousel with smooth transitions and a clean dark UI.
The app includes a player model, animated home screen, and a detailed player profile screen with stats.

Features
Full-screen swipable player cards
Smooth hero animations between screens
Player info card at the bottom of each page
Detailed player screen with extended information
Simple, clean, and easy-to-read code structure
Loads player images directly from network links
