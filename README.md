## Football Player Profiles-UI ⚽️
A small modern animated Flutter app that displays football players in a full-screen carousel with smooth transitions and a clean dark UI.
The app includes a player model, animated home screen, and a detailed player profile screen with stats.

## Features

1. Full-screen swipable player cards  
2. Smooth hero animations between screens  
3. Player info card at the bottom of each page  
4. Detailed player screen with extended information  
5. Simple, clean, and easy-to-read code structure  
6. Loads player images directly from network links
